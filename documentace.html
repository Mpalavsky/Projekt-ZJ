1. Úvod

XML které zde je popsáno slouží jako konfigurační soubor pro smartphone.

2. Metodika

Zdroje: jako zdroje jsem použil přednášky, znalosti a zápisky získané na cvičení a prezentací z Moodlu a dále knihu XML pro každého ISBN 80-7169-860-1

Aplikace: Využití prostředí Microsoft Visual Studio Code a NetBeans na ověření schémat.

6. Transformace

Zde popíšu jednotlivé transformace a jaké jsem pro ně použil příkazy

7. GIT

GIT jsem použil jako nástroj pro uložení projektu a na jiném účtu ho používám pro zálohu projektů

GIT projekt

1. Network

První část transformace vypíše nastavení sítě(network) Profilu2 a hodnoty true, false přepíše na pro člověka srozumitelnější hodnoty. Použil jsem select, if

2. Region

Druhý část transformace vypíše nastavení Region pro všechny profily. Použil jsem select, if, foreach.

3. Sound

Třetí část transformace seřadí hodnoty hlasitosti masterVolume od nejmenšího po největší a u hodnot 80 až 100 vypíše hlášku s varováním. Použil jsem select, if, foreach, sort.

8. Závěr

Jazyk XML bych použil jako vstupní soubor při konfiguraci telefonu, případně aplikace. Výhodou je přehlednost, jednoznačnost a snadná editace v jakémkoli textovém editoru
